# 📝 CatatanToDoApp

> Aplikasi Android untuk mengelola catatan dan todo list dengan fitur gambar dan pengingat waktu

## ✨ Fitur

- 📷 **Catatan dengan Gambar** - Tambahkan foto ke dalam catatan
- ⏰ **Pengingat Waktu** - Set waktu untuk todo list
- ✅ **Status Checklist** - Centang todo yang sudah selesai
- 💾 **Penyimpanan Lokal** - Data tersimpan di perangkat

## 📱 Kebutuhan Sistem

- Android 5.0 ke atas
- Akses kamera untuk foto
- Akses penyimpanan untuk gambar

## 🔧 Instalasi

1. **Buka di Android Studio**
   - Buka Android Studio Ladybug
   - File → Open → Pilih folder project
   - Tunggu hingga selesai loading

2. **Run Aplikasi**
   - Hubungkan device Android atau gunakan emulator
   - Klik tombol "Run"

## 📖 Cara Penggunaan

### 📝 Membuat Catatan
1. Tap tombol tambah
2. Tulis catatan Anda
3. Tambahkan gambar jika diperlukan
4. Simpan

### ✅ Mengelola Todo
1. Masuk ke bagian Todo
2. Tambah todo baru
3. Set waktu pengingat
4. Centang jika sudah selesai
